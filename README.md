# Chronogrid

Chronogrid is a calendar and planner you run on your own computer. You make events, drag them around to move them, and switch between day, week, month and a few other views. It also shows you where your time actually went during the week. Everything saves on your own device, so there's no login and it works without internet.

To try it run npm install and npm start to get it as a desktop app then it will be ready for full use.

## How to use it

The first time you open it there's a quick onboarding that points everything out. After that you make an event by clicking Create or just dragging on the grid. Drag an event around to move it to a different time or day. Jump between dates with the little calendar or the arrows up top, and switch views with the tabs on the left. Everything you do saves on its own, so you can close it and come back later.

## Features

- Day, week, work week, month and year views
- A week grid like a real calendar with a live line for the current time
- Drag and drop to move events around
- Board and schedule views for planning things out
- Analytics that break down where your time went
- Focus timers that can chain into a rest timer
- Light and dark mode
- Time zone picker
- Import and export .ics files
- Saves everything locally, no account or internet needed

I built this myself and used Claude minimally for small things.
